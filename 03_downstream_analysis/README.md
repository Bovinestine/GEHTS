The pipeline of analysis and visualization of preprocessed data is composed of 4 code flies, each for a main figure of GEHTS-chip paper.

'./figure1.r' contains a function for drawing a dot plot of platform comparison using "./Transcriptomic_Screening_Platform_Comparison.csv" which is provided as a supplementary data of GEHTS-chip paper.

'./figure2.r' contains functions for fig.2c~h.

'./figure3.r' contains functions for fig.3b~i.

'./figure4.r' contains functions for fig.4b~f.
